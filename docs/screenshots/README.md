# 截图与演示素材

请将 Web 界面截图、GIF 动图放在本目录，用于 README 与答辩展示。

## 建议截图

| 文件名 | 内容 |
|--------|------|
| `web_chat.png` | 主对话区 + 一次完整问答 |
| `web_tools_panel.png` | 右侧工具调用链路面板 |
| `cli_trace.png` | CLI 终端 + trace 输出（可选） |
| `demo.gif` | 从提问到工具调用再到回复的短动图（可选） |

## 占位结构

```
docs/screenshots/
  README.md          ← 本说明
  web_chat.png       ← 待添加
  web_tools_panel.png
  demo.gif           ← 可选
```

截图后可在 README 中引用：

```markdown
![Web 对话界面](docs/screenshots/web_chat.png)
```
